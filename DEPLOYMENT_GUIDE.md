## Method 2: GitHub Pages (Requires GitHub Account)

### Steps:

1. **Create GitHub Account** (if you don't have one):
   - Go to https://github.com/signup
   - Sign up for free

2. **Create New Repository**:
   - Click "+" icon ‚Üí "New repository"
   - Name: `pokemon-cards-store`
   - Make it Public
   - Click "Create repository"

3. **Upload Files**:
   - Click "uploading an existing file"
   - Drag all files from `C:\Users\khana\Desktop\pokemon-cards-store`
   - Click "Commit changes"

4. **Enable GitHub Pages**:
   - Go to repository Settings
   - Scroll to "Pages" section
   - Source: Select "main" branch
   - Click "Save"

5. **Get Your URL**:
   - Wait 2-3 minutes
   - Your site will be live at:
   - `https://YOUR-USERNAME.github.io/pokemon-cards-store`
