1.1 THE THREE PILLARS OF WEB DEVELOPMENT
-----------------------------------------

Every website is built using three core technologies:

A) HTML (HyperText Markup Language)
   - Purpose: Structure and content
   - What it does: Defines what appears on the page
   - Example: Headings, paragraphs, images, buttons
   - Think of it as: The skeleton of your website

B) CSS (Cascading Style Sheets)
   - Purpose: Styling and layout
   - What it does: Makes things look beautiful
   - Example: Colors, fonts, spacing, animations
   - Think of it as: The skin and clothes of your website

C) JavaScript
   - Purpose: Interactivity and logic
   - What it does: Makes things work and respond
   - Example: Button clicks, form validation, dynamic content
   - Think of it as: The brain and muscles of your website

1.2 HOW THEY WORK TOGETHER
---------------------------

Example: A "Add to Cart" button

HTML:
```html
<button id="add-to-cart">Add to Cart</button>
```
This creates the button structure.

CSS:
```css
#add-to-cart {
    background-color: #e74c3c;
    color: white;
    padding: 10px 20px;
    border-radius: 5px;
}
```
This makes the button look attractive.

JavaScript:
```javascript
document.getElementById('add-to-cart').addEventListener('click', function() {
    // Add product to cart
    alert('Product added to cart!');
});
```
This makes the button DO something when clicked.

1.3 CLIENT-SIDE VS SERVER-SIDE
-------------------------------

Your website is CLIENT-SIDE (runs in the browser):
- No server needed for basic functionality
- Data stored in browser (localStorage)
- Fast and responsive
- Works offline (after first load)

Advantages:
‚úÖ Free hosting (GitHub Pages)
‚úÖ Fast performance
‚úÖ Easy to maintain
‚úÖ No server costs

Limitations:
‚ùå Data stored locally (not shared between devices)
‚ùå No real database
‚ùå Limited security for sensitive data

4.1 WHAT IS LOCAL STORAGE?
---------------------------

localStorage is a browser feature that stores data permanently:
- Stores key-value pairs
- Data persists even after closing browser
- Limit: ~5-10MB per domain
- Only stores strings (must convert objects)

4.2 HOW WE USE IT
-----------------

In data.js, we created a dataManager object:

```javascript
const dataManager = {
    // Initialize with default products
    init: function() {
        if (!localStorage.getItem('products')) {
            this.saveProducts(defaultProducts);
        }
    },
    
    // Get all products
    getProducts: function() {
        const data = localStorage.getItem('products');
        return data ? JSON.parse(data) : [];
    },
    
    // Save products
    saveProducts: function(products) {
        localStorage.setItem('products', JSON.stringify(products));
    }
};
```

Key Concepts:

A) JSON.stringify()
   - Converts JavaScript object to string
   - Example: {name: "Pikachu"} ‚Üí '{"name":"Pikachu"}'
   - Why? localStorage only stores strings

B) JSON.parse()
   - Converts string back to JavaScript object
   - Example: '{"name":"Pikachu"}' ‚Üí {name: "Pikachu"}
   - Why? We need objects to work with

5.1 HOW THE CART WORKS
----------------------

Step-by-step process:

1. User clicks "Add to Cart" on a product
2. JavaScript function addToCart() is called
3. Function checks if product already in cart
4. If yes: Increase quantity
5. If no: Add new item with quantity 1
6. Save updated cart to localStorage
7. Update cart count in header
8. Show success message

5.2 ADD TO CART CODE EXPLAINED
-------------------------------

```javascript
function addToCart(productId, quantity = 1) {
    // Get current cart from localStorage
    const cart = dataManager.getCart();
    
    // Check if product already in cart
    const existingItem = cart.find(item => item.productId === productId);
    
    if (existingItem) {
        // Product exists, increase quantity
        existingItem.quantity += quantity;
    } else {
        // New product, add to cart
        cart.push({
            productId: productId,
            quantity: quantity
        });
    }
    
    // Save updated cart
    dataManager.saveCart(cart);
    
    // Update UI
    updateCartCount();
    showToast('Added to cart!', 'success');
}
```

Key JavaScript Concepts:

A) Array.find()
   - Searches array for matching item
   - Returns first match or undefined
   ```javascript
   const item = cart.find(item => item.productId === 5);
   ```

B) Arrow Functions
   - Short syntax for functions
   ```javascript
   item => item.productId === 5
   // Same as:
   function(item) {
       return item.productId === 5;
   }
   ```

C) Array.push()
   - Adds item to end of array
   ```javascript
   cart.push({productId: 5, quantity: 1});
   ```

5.4 QUANTITY UPDATES
--------------------

```javascript
function updateQuantity(productId, newQuantity) {
    // Don't allow quantity less than 1
    if (newQuantity < 1) {
        if (confirm('Remove this item?')) {
            removeFromCart(productId);
        }
        return;
    }
    
    // Get product to check stock
    const product = dataManager.getProductById(productId);
    
    // Don't allow more than available stock
    if (newQuantity > product.stock) {
        showToast('Not enough stock!', 'error');
        return;
    }
    
    // Update cart
    const cart = dataManager.getCart();
    const item = cart.find(i => i.productId === productId);
    item.quantity = newQuantity;
    
    // Save and refresh
    dataManager.saveCart(cart);
    loadCart();
}
```

7.5 CREATING ORDER
------------------

```javascript
createOrder: function(customerInfo) {
    // Get cart items
    const cart = this.getCart();
    
    // Create order object
    const order = {
        id: Date.now(),  // Unique ID using timestamp
        date: customerInfo.date,
        customer: customerInfo,
        items: cart.map(item => {
            const product = this.getProductById(item.productId);
            return {
                productId: item.productId,
                name: product.name,
                price: product.price,
                quantity: item.quantity
            };
        }),
        total: customerInfo.totalAmount,
        status: 'completed'
    };
    
    // Save order
    const orders = this.getOrders();
    orders.push(order);
    this.saveOrders(orders);
    
    // Update stock
    cart.forEach(item => {
        const product = this.getProductById(item.productId);
        product.stock -= item.quantity;
    });
    this.saveProducts(this.getProducts());
    
    // Clear cart
    this.clearCart();
    
    return { success: true, order: order };
}
```

Key Concepts:

A) Date.now()
   - Returns current timestamp
   - Used as unique order ID

B) Array.map()
   - Creates new array by transforming each item
   - Doesn't modify original array

C) Stock Management
   - Reduces stock after successful order
   - Prevents overselling

```javascript
function saveProduct(event) {
    event.preventDefault();
    
    // Get form values
    const product = {
        id: document.getElementById('product-id').value || Date.now(),
        name: document.getElementById('product-name').value,
        price: parseFloat(document.getElementById('product-price').value),
        stock: parseInt(document.getElementById('product-stock').value),
        marketPrice: parseFloat(document.getElementById('product-market-price').value),
        description: document.getElementById('product-description').value,
        image: currentImageData || 'default-image.jpg',
        marketUrl: document.getElementById('product-market-url').value,
        marketSource: document.getElementById('product-market-source').value
    };
    
    // Get existing products
    const products = dataManager.getProducts();
    
    // Check if editing or adding
    const existingIndex = products.findIndex(p => p.id == product.id);
    
    if (existingIndex !== -1) {
        // Update existing
        products[existingIndex] = product;
        showToast('Product updated!', 'success');
    } else {
        // Add new
        products.push(product);
        showToast('Product added!', 'success');
    }
    
    // Save and refresh
    dataManager.saveProducts(products);
    loadProductsTable();
    cancelForm();
}
```

Key Concepts:

A) parseFloat() / parseInt()
   - Converts string to number
   - parseFloat: Decimal numbers
   - parseInt: Whole numbers

B) Array.findIndex()
   - Returns index of matching item
   - Returns -1 if not found

C) Ternary Operator
   ```javascript
   const id = existingId || Date.now();
   // Same as:
   const id = existingId ? existingId : Date.now();
   ```

8.3 IMAGE HANDLING

```javascript
function previewImage(event) {
    const file = event.target.files[0];
    
    if (file) {
        const reader = new FileReader();
        
        reader.onload = function(e) {
            currentImageData = e.target.result;
            const preview = document.getElementById('image-preview');
            preview.innerHTML = `<img src="${currentImageData}" alt="Preview">`;
        };
        
        reader.readAsDataURL(file);
    }
}
```

Key Concepts:

A) FileReader API
   - Reads file contents
   - Converts to base64 string
   - Can be stored in localStorage

B) readAsDataURL()
   - Converts file to data URL
   - Format: data:image/png;base64,iVBORw0KG...
   - Can be used directly in <img src="">

C) Event Handling
   - reader.onload: Called when file loaded
   - e.target.result: Contains file data

11.1 CODE ORGANIZATION
----------------------

A) DRY Principle (Don't Repeat Yourself)
```javascript
// Bad - Repeated code
function addProduct() {
    const products = JSON.parse(localStorage.getItem('products'));
    // ...
}

function deleteProduct() {
    const products = JSON.parse(localStorage.getItem('products'));
    // ...
}

const dataManager = {
    getProducts: function() {
        return JSON.parse(localStorage.getItem('products'));
    }
};
```

B) Separation of Concerns
- HTML: Structure only
- CSS: Styling only
- JavaScript: Logic only

C) Naming Conventions
```javascript
// Variables: camelCase
const productName = "Charizard";

const MAX_QUANTITY = 10;

function getProducts() {}
function addToCart() {}

// Classes: PascalCase
class ProductManager {}
```
